A general introduction to MongoDB

MongoDB is a database sure, okay cool so I can store data on a computer. That's nice, but why would I use MongoDB? Well, MongoDB uses a different concept of database technology. It uses this idea of NoSQL. In the past MySQL was by far the most used database technology. Using SQL however comes with a few things that you have to keep in mind. Different parts of your data should have relationships between each other. It becomes very complicated storing none relational data inside tables and therefore SQL becomes overly complicated for what you are trying to accomplish and that is just storing data.

Connecting to MongoDB

In this section I assume you already have MongoDB installed onto your system that you are currently working with. If you need help installing MongoDB you can read into their documentation. I have found it easy to follow. Learn to install MongoDB.

To start up mongo and be able to connect to it you have to first open a terminal window. There are different ways to do this depending on the system you are using. Now once you have the terminal open you can start by typing into your terminal window the command 

mongo
.
